Here you will find some very useful HTML tags to utilize in your work, Each tag is followed by an example to help you understand the tag better.


<h1> - This HTML tag is used for heading.

<h1> This is Heading 1 </h1>


<p> - This HTML tag is used for writing a paragraph.

<p> This Is My Paragraph, I can type any information here </p>


<em> - This HTML tag is used to emphasize the required text.

<em> This Text Is emphasized </em>


<b> - This HTML tag is used to convert any text within it to bold.

<b> This Text Is Bold </b>


<i> - This HTML tag is used to convert any text within it to italic.

<i> This Text Is In Italic Font </i>


<ttle> - This HTML tag is used to give your webpage is title.

<title> This Is The Page Title </title>


<img src> - This HTML tag is used to insert an image in the webpage.

<img src = " Insert Your Image Link here " />


<audio src> : - This HTML tag is used to inset audio in your webpage.

<audio src = " Insert Your Audio Link Here " </audio>


<br/> - This HTML tag adds a line break to your webpage.

This Is An <br/>
Example For Line Break


<strong> - This HTML tag that is used to highlight important text.

<strong> This Text Is Very Important </strong>


<video src> - This HTML tag is used to inset video in your webpage.

<video = " Insert Your Video Link Here" </video>
